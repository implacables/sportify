Example layout after runs:

```
results/
├── soccernet-gsr/
│   └── 20260524T120000Z/
│       ├── run-meta.json
│       ├── manifest.yaml
│       └── stdout.log
└── throughput/
    └── 20260524T140000Z/
        ├── run-result.json
        └── job.meta.json
```

Commit summaries or charts to docs if needed for thesis; do not commit raw logs or prediction zips.
